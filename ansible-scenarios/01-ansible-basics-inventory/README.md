# Ansible Basics - Inventory Management

## Overview
Learn Ansible fundamentals by creating inventories, organizing hosts into groups, and testing connectivity with ad-hoc commands.

## What You'll Learn
- Static vs dynamic inventory concepts
- Host grouping and patterns
- Ansible variables and host-specific configurations
- Ad-hoc command execution
- Testing connectivity with ping module

## Prerequisites
- Ansible installed locally
- Docker installed (for target hosts)
- Basic understanding of YAML

## Resources Created
- Static inventory file with localhost and Docker containers
- Docker containers: web1, web2, db1 (Ubuntu)
- Host and group variables
- ansible.cfg configuration

## Cleanup
Run the cleanup commands to stop and remove all Docker containers created in this scenario.
