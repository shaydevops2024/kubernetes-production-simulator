# Package Management with Ansible

## Overview
Learn to manage packages across different Linux distributions using Ansible's package modules and ensure services are properly configured.

## What You'll Learn
- Using apt module for Ubuntu/Debian
- Installing and removing packages
- Managing package repositories
- Ensuring services are started and enabled
- Handling package dependencies

## Prerequisites
- Ansible installed locally
- Docker installed (for target hosts)
- Completed scenario 01 (inventory basics)

## Resources Created
- Docker containers: ubuntu-pkg, debian-pkg (Ubuntu/Debian targets)
- Installed packages: nginx, curl, htop
- Running nginx service
- Custom apt repository configuration

## Cleanup
Run the cleanup commands to stop and remove all Docker containers and configurations created in this scenario.
