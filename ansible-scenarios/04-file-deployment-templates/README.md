# File Deployment and Jinja2 Templates

## Overview
Learn to deploy configuration files using Ansible templates, variables, and facts. Master the art of dynamic configuration management.

## What You'll Learn
- Using Jinja2 templates with variables
- Ansible facts and how to use them
- File permissions and ownership
- Template conditionals and loops
- Configuration validation before deployment

## Prerequisites
- Ansible installed locally
- Docker installed (for target hosts)
- Basic understanding of Jinja2 templating

## Resources Created
- Docker containers: web1, web2 (nginx servers)
- nginx configuration files from templates
- Application config files with environment-specific values
- Validated configurations

## Cleanup
Run the cleanup commands to stop and remove all Docker containers created in this scenario.
