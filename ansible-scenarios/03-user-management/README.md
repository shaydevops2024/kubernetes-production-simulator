# User and Permission Management

## Overview
Learn to manage Linux users, groups, SSH keys, and sudo permissions using Ansible automation.

## What You'll Learn
- Creating users and groups
- Managing SSH key-based authentication
- Configuring sudo access
- Setting up user environments (.bashrc, .bash_profile)
- File ownership and permissions

## Prerequisites
- Ansible installed locally
- Docker installed (for target hosts)
- Basic Linux user management concepts

## Resources Created
- Docker containers: server1, server2 (Ubuntu targets)
- Users: devuser, dbadmin, appuser
- Groups: developers, dbadmins
- SSH keys for passwordless authentication
- Sudo configuration for specific users

## Cleanup
Run the cleanup commands to stop and remove all Docker containers created in this scenario.
