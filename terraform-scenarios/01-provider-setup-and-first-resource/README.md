# Provider Setup and First Resource
Learn Terraform fundamentals by configuring the AWS provider and creating your first S3 bucket with versioning and tags.

## Learning Objectives
- Understand Terraform provider configuration and authentication
- Learn the terraform init, plan, and apply workflow
- Create an S3 bucket with versioning enabled and proper tagging
- Inspect Terraform state to understand how infrastructure is tracked

## Prerequisites
- Basic understanding of cloud infrastructure concepts
- Familiarity with CLI tools and JSON/HCL syntax

## Resources Created
- AWS provider configuration block
- S3 bucket with versioning enabled
- Resource tags for cost tracking and organization

## Cleanup
The final step removes all displayed configuration files and resets the scenario environment.
