# Rolling Updates and Rollbacks

## Objective
Learn to perform zero-downtime deployments and rollback to previous versions safely.

## What You'll Learn
- Perform rolling updates
- Monitor rollout status
- Pause and resume rollouts
- Rollback to previous versions
- Understand rollout strategies

## Duration
20 minutes

## Success Criteria
- Successfully roll out new version
- Zero downtime during update
- Successfully rollback when needed
- All pods remain healthy throughout
