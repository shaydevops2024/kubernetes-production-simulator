# RBAC Setup and Testing

## Objective
Implement Role-Based Access Control to demonstrate least-privilege access principles.

## What You'll Learn
- Create Roles and ClusterRoles
- Bind roles to users and service accounts
- Test permissions
- Debug RBAC issues
- Implement least-privilege

## Duration
30 minutes

## Success Criteria
- Roles created correctly
- RoleBindings work as expected
- Permissions properly restricted
- Unauthorized access denied
- Debug commands mastered
