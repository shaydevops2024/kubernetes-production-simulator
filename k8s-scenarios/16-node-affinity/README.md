# Node Affinity and Taints

## Objective
Control pod placement using node affinity, node selectors, and taints/tolerations.

## What You'll Learn
- Use node selectors
- Configure node affinity
- Apply taints to nodes
- Add tolerations to pods
- Understand scheduling constraints

## Duration
25 minutes

## Success Criteria
- Pods scheduled according to affinity
- Taints prevent unwanted scheduling
- Tolerations allow specific pods
- Node labels properly configured
- Scheduling behavior verified
