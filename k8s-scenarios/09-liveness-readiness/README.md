# Liveness and Readiness Probes

## Objective
Configure health checks to ensure pods are healthy and ready to serve traffic.

## What You'll Learn
- Configure liveness probes
- Set up readiness probes
- Understand probe types (HTTP, TCP, Exec)
- Set appropriate thresholds
- Debug probe failures

## Duration
25 minutes

## Success Criteria
- Probes configured correctly
- Unhealthy pods automatically restarted
- Traffic only sent to ready pods
- No false-positive failures
