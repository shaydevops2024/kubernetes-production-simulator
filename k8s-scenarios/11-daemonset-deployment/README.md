# DaemonSet Deployment

## Objective
Deploy and manage DaemonSets to run pods on every node in the cluster.

## What You'll Learn
- Create DaemonSets
- Understand DaemonSet use cases
- Update DaemonSets
- Handle node selectors
- Monitor DaemonSet status

## Duration
15 minutes

## Success Criteria
- DaemonSet runs on all nodes
- Pods automatically added to new nodes
- Updates roll out correctly
- No scheduling conflicts
