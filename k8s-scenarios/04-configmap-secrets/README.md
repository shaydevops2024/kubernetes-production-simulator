# ConfigMap & Secrets Management

## Objective
Master ConfigMaps and Secrets for managing application configuration and sensitive data.

## What You'll Learn
- Create and manage ConfigMaps
- Work with Kubernetes Secrets
- Mount configs as environment variables
- Mount configs as volumes
- Update configurations without downtime

## Duration
15 minutes

## Success Criteria
- ConfigMap created and mounted
- Secret created with proper encoding
- Application reads configuration correctly
- Live reload works for config changes
