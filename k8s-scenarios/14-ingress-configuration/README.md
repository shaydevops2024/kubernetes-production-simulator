# Ingress Configuration & Service Debugging

## Objective
Configure Ingress for external access and compare different service access patterns.

## What You'll Learn
- Configure Ingress resources
- Set up path-based routing
- Compare NodePort vs ClusterIP vs Ingress
- Debug service connectivity
- Implement SSL/TLS termination

## Duration
25 minutes

## Success Criteria
- Ingress routes traffic correctly
- Path-based routing works
- Service types understood
- Debug tools mastered
- External access functional
