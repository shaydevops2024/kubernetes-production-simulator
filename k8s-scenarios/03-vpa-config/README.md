# Vertical Pod Autoscaler (VPA) Configuration

## Objective
Learn how to configure and use Vertical Pod Autoscaler to automatically adjust CPU and memory requests based on actual usage.

## What You'll Learn
- Install and configure VPA
- Set up VPA recommendations
- Apply VPA policies to deployments
- Monitor resource request adjustments
- Understand VPA update modes

## Duration
25 minutes

## Success Criteria
- VPA installed and running
- VPA provides resource recommendations
- Pod requests automatically adjusted
- Application remains stable during updates
