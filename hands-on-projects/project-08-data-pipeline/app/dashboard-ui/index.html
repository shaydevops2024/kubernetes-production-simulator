What is Consumer Lag?

Consumer lag is the difference between the latest offset produced to a Kafka partition and the committed offset of the consumer group.

A lag near 0 means the Spark processor is keeping up with the producer. Growing lag means it's falling behind — try pausing the producer (curl -X POST http://localhost:8090/control/pause) and watch the lag drain to zero, then resume (/control/resume) to see it spike momentarily before recovering.
